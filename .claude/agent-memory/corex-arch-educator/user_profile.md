Go 백엔드 개발자. Core-X 프로젝트를 분산 KV 스토어 학습 프로젝트로 직접 구축 중이며, ADR을 통해 설계 결정을 명시적으로 기록하는 방식으로 작업한다.

Phase 단위로 기능을 점진적으로 구현하며, 각 Phase마다 구현 계획 문서(docs/superpowers/plans/)를 먼저 작성하고 코드를 작성한다.

Claude Code와 협업 시 교육적 설명(왜 이렇게 설계했는가)을 선호하고, 실제 코드와 ADR을 연결한 분석을 원한다.
